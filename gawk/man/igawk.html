名前

igawk - インクルードファイルを用いる gawk

書式

igawk [ all gawk options ] -f program-file [ -- ] file ...
igawk [ all gawk options ] [ -- ] program-text file ...

説明

igawk は、 gawk(1) に「インクルードファイル」機能を追加する、簡素なシェルスクリプトである。

igawk で扱う AWK プログラムは gawk と同じであるが、カレントディレクトリもしくは検索パスディレクトリにあるファイル getopt.awk をインクルードする為に、

@include getopt.awk

のような行をプログラムに書くことが可能である。

オプション

AWK 言語および gawk がサポートするオプションに関する完全な説明は、 gawk(1) を参照すること。

例

cat << EOF > test.awk
@include getopt.awk
BEGIN {
	while (getopt(ARGC, ARGV, "am:q") != -1)
		...
}
EOF
igawk -f test.awk
